# AI Chat — тестовое задание

Рабочий клиентский AI-чат на Next.js (App Router) + TypeScript + Tailwind CSS.
Без бэкенда и API-ключей: ответы ассистента — заготовленные моки, всё
состояние живёт в React state. Задание для кандидата — в
[ASSIGNMENT.md](./ASSIGNMENT.md).

Что уже работает: отправка сообщений с эффектом печатания, переключение
и создание диалогов, живой поиск, выбор модели, тумблер «Опираться на
материалы», копирование сообщений с тостом, раскрытие «Источников»,
автоскролл с кнопкой «Вернуться вниз».

## Требования

- Node.js **18+** (рекомендуется 20+)
- npm

## Установка и запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:3000 — увидите работающий чат.

Продакшн-сборка:

```bash
npm run build
npm start
```

## Структура

```
app/
  layout.tsx           # корневой layout: шрифт Manrope, подключение globals.css
  page.tsx             # страница чата (/) — рендерит ChatApp
  globals.css          # дизайн-токены (CSS-переменные) + Tailwind
  icon.tsx             # favicon (генерируется через next/og)
components/
  ChatApp.tsx          # корневой клиентский компонент: всё состояние чата
  Sidebar.tsx          # сайдбар: новый диалог, поиск, проекты, список чатов,
                       # карточка модели, строка пользователя
  ChatHeader.tsx       # шапка: название чата, активная модель, тумблер материалов
  MessageList.tsx      # лента: автоскролл, «Вернуться вниз», пустое состояние
  Message.tsx          # одно сообщение (user / assistant, блок «Источники»)
  TypingIndicator.tsx  # пузырь «ассистент печатает…»
  EmptyState.tsx       # пустой диалог: примеры запросов
  Composer.tsx         # поле ввода: авторазмер, Enter / Shift+Enter
  ModelSelect.tsx      # карточка выбора модели в сайдбаре с dropdown
  Toast.tsx            # простое тост-уведомление
  icons.tsx            # инлайн SVG-иконки
lib/
  mock-data.ts         # мок-данные: чаты, проекты, сообщения, модели, пул ответов
```

Интерактив целиком клиентский: состояние держит `components/ChatApp.tsx`,
данные инициализируются из `lib/mock-data.ts`. Реальных API-вызовов
и сохранения между перезагрузками нет — это сделано намеренно.
